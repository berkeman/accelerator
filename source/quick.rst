Quickstart
==========


Install

.. code-block::

   pip install accelerator


Get help

.. code-block::

   ax help


Start a new project

.. code-block::

   ax init <projectname>


Show available method directories, methods and descriptions

.. code-block::
   
  ax method

  
Show build scripts and descriptions

.. code-block::

   ax script

   
Run a build script named "``build_myprogram.py``"

.. code-block::

   ax run myprogram
